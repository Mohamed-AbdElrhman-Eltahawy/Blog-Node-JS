# Blog API

REST API for a blog platform built with Express and TypeScript, including JWT-based authentication, post CRUD endpoints, and image uploads with Cloudinary.

## Description

This project provides backend APIs for user authentication and blog post management. It uses MongoDB with Mongoose for persistence and secures protected routes with token-based auth middleware.
